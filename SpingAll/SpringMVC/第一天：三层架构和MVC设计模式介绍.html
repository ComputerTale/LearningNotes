RequestMapping注解

作用：建立请求URL和处理请求方法之间的对应关系

/**
 * @author fangjie
 * @version 1.0
 * @date 2020/2/24 17:36
 * @description 项目控制器
 *
 * @RequestMapping
 *  作用：建立请求URL和处理请求方法之间的映射关系
 *  添加位置：类和方法
 *  写在类上：方便模块化请求
 *  写在方法上：建立映射关系
 *  属性：
 *      value：指定请求的URL
 *      path：等同于value
 *      method：用于指定请求的方式
 *          举例：
 *              method = {RequestMethod.POST}
 *      params：用于指定请求参数的条件，要求请求参数的key和value必须和配置的一模一样
 *          举例：
 *          param = {"key>100"}
 * 请求：user/param?key=150
 *   看起来满足了要求，但是是错误的请求，参数必须一样
 *   正确请求：user/param?key>100
 *      headers：用于指定限制请求消息头
 *          举例：
 *              headers = {"Accept"}
 *  以上属性有两个及以上参数，参数是并列的
 */
@Controller
@RequestMapping(path = "/user",method = {RequestMethod.GET},params = {"username","password=123"})
public class HelloController {
    /**
     * 该注解对应的请求是localhost:8080/项目名/user/hello
     * @return
     */
    @RequestMapping(path = "/hello")
    public String sayHello(){
        System.out.println("Hello StringMVC");
        return "success";
    }
}

/**
 * @author fangjie
 * @version 1.0
 * @date 2020/2/25 16:52
 * @description 自定义类型转换
 *
 * 实现Spring框架提供的Converter接口
 * Converter<S,T>
 *     该接口存在两个泛型
 *      S：表示传入的字符串
 *      T：表示要转的类型
 */
public class StringToDateConverter implements Converter<String,Date> {
    @Override
    public Date convert(String s) {
        if(s == null){
            throw new RuntimeException("请传入参数");
        }
        try {
            DateFormat df = new SimpleDateFormat("yyyy-mm-dd");
            return df.parse(s);
        } catch (ParseException e) {
            throw new RuntimeException("类型转换错误");
        }
    }
}

    <property name="converters">
        <set>
            <bean class="cn.itcast.utils.StringToDateConverter"/>
        </set>
    </property>
</bean>
<!-- 开启SpringMVC框架注解支持，配置类型转换器-->
<mvc:annotation-driven conversion-service="conversionService"/>

常用注解

1. RequestParam注解
作用：把请求中的指定名称的参数传递给控制器中的形参赋值
省略该注解，作用等同于以变量名作为value，required为false
属性：

value：请求参数中的名称
required：请求参数中是否必须提供此参数（value中的参数），默认值是true，必须提供
defaultValue：默认参数值，如果设置了该值，required=true将失效，自动为false,如果没有传该参数，就使用默认值


2. RequestBody注解
作用：获取请求体（字符串）的内容（get方式不使用，因为get方法没有请求体），得到是key=value&key=value...结构的数据
属性：

required：是否必须有请求体，默认值是true
