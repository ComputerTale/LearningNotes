forms表单
使用v-model来双向绑定输入框input
<input
type="text"
id="email"
class="form-control"
v-model="userData.email">
这里v-model会实时监控输入框中的内容
设置v-model的属性
lazy:当用户输入完毕时才进行处理，即只获取当前输入的最后结果
trim：用于去除开头和结尾的空格
number：把输入的内容强制转换为数值类型，比如获取浮点数
对于textarea标签，也是使用v-model来插入值，而不是在标签内插值
<!-- Interpolation between <textarea>{{ test }}</textarea> doesn't work!-->
<textarea
id="message"
rows="5"
class="form-control"
v-model="message"></textarea>
通过textarea传输的内容是可以保持换行特征的，不过在输出的时候要设置好换行的样式
<!-- 保留多行文本中的换行 -->
<p style="white-space:pre">Message:{{ message }}</p>
复选框checkbox
使用一组复选框，可以直接用一个数组表示即可
数组的绑定：Vue会自动将传入的参数加入数组，删除参数，Vue也会自动将该参数从数组中删除
<!-- 复选键，普通邮件还是信息邮件 -->
<label for="sendmail">
<input
type="checkbox"
id="sendmail"
value="SendMail"
v-model="sendMail"> Send Mail
</label>
<label for="sendInfomail">
<input
type="checkbox"
id="sendInfomail"
value="SendInfoMail"
v-model="sendMail"> Send Infomail
</label>

sendMail:[]
单选框radio
当几个单选框绑在同一个变量上时，Vue会自动将它们放在同一组中，这样每次只能选择其中一个，同时，Vue会将被选中的单选框的值赋给变量
<label for="male">
<input
type="radio"
id="male"
value="Male"
v-model="gender"> Male
</label>
<label for="female">
<input
type="radio"
id="female"
value="Female"
v-model="gender"> Female
</label>

gender:'Male'
下拉菜单select，option表示选项
<label for="priority">Priority</label>
<select
id="priority"
class="form-control"
v-model="seletedPriority">
<option
v-for="priority in priorities" :key="priority"
:selected="priority == 'Medium'">
{{ priority }}
</option>
</select>

seletedPriority:'High',
priorities:['High','Medium','Low']
设置下拉菜单初始值的两种方式：
一、在选项中添加被选择属性，值为Ture和False
<option
v-for="priority in priorities" :key="priority"
:selected="priority == 'Medium'">
{{ priority }}
</option>
二、为下拉菜单绑定数据
<select
id="priority"
class="form-control"
v-model="seletedPriority">

seletedPriority:'High',
注意：当使用了第二种方式以后，第一种方式会失效，因此直接采取第二种方式即可
